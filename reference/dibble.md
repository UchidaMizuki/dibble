# Build a dimensional data frame

`dibble()` constructs a dimensional data frame called a dibble.

## Usage

``` r
dibble(..., .dim_names = NULL)
```

## Arguments

- ...:

  A set of name-measure pairs.

- .dim_names:

  A list of dimension names.

## Value

A dibble.

## Details

Manipulation functions:

- mutate()

- rename()

- select() & relocate()

- slice()
